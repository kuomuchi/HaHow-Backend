# HaHow 專案

## 說明
這是一個小專案，此專案的需要從HaHow的API抓取資料，然後用裡面的資料製作做一個API，讓使用可以透過此專案的API抓取HaHow API的資料。

最基本的功能為抓取創建API。

會再透過此API延伸更多的功能。此API非彼API。

## 如何啟動
```
npm install
node index.js
```

## 未來功能
* 加入docker
* 新增偽API資料
* 新增cache / 定期更新HaHow API資料。
* 加入DB，NoSQL
